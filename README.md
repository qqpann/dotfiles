# dotfiles

## Manual installation

- VSCode
- iTerms
- GitHub Desktop
- Homebrew (macOS)

## iTerms theme setup

- General > Window > Uncheck "Native full screen windows"
- Profiles > Window
  - Style: set to "Full Screen"
  - Screen: set to "Screen with cursor"
  - Space: set to "All spaces"

Profiles can also be imported/exported as JSON.
